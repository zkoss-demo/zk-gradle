# ZK Gradle

Example ZK project using gradle

## Useful Build Commands

run jetty (with gretty plugin)
```
./gradlew appRun
```
run jetty (with jetty-runner - faster startup time)
```
./gradlew startJettyRunner
```
http://localhost:8080/zk-gradle

run tests
```
./gradlew test
```

build war
```
./gradlew war
```
